# 🛠️ Technology Stack Detail

## Backend & API
*   **Python 3.12**: The core programming language.
*   **FastAPI**: A modern, high-performance web framework for building APIs.
*   **Uvicorn**: An ASGI web server implementation for Python.

## Artificial Intelligence
*   **XGBoost**: An advanced gradient boosting library used to build the high-accuracy prediction model.
*   **Scikit-Learn**: Used for data preprocessing and feature scaling.
*   **OpenAI (GPT-4o)**: Powers the natural language generation for customer emails.

## Data & Storage
*   **PostgreSQL**: A powerful, open-source relational database for enterprise-grade data storage.
*   **Pandas & NumPy**: For efficient data manipulation and numerical calculations.

## Frontend & Monitoring
*   **Streamlit**: Used to create the interactive data dashboard.
*   **Plotly**: Powers the dynamic charts and visualizations in the dashboard.

## Cloud & Utilities
*   **smtplib (SSL)**: Secure standard library for sending transactional emails directly via SMTP (bypassing the need for paid API services).
*   **Python-Dotenv**: Manages sensitive credentials (API keys, DB passwords) securely.
*   **Python-Docx**: Used to generate automated Word reports.
